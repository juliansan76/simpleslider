Welcome to SimpleSlider

An image slider written in plain javascript

Slider of 120px image width:

Include simpleslider.js, and simpleslider.css

Paste this html code at any place of your page:

<div id="sld[N]" class="sld-cluster">
</div>

Where [N] is an integer that represents the index of the sliders objects array sldsObjs[], which holds the following item structure:

{ "sldItemWide" : 120, // width of each item in px
 "sldItemPosition" : 0, //
 "sldItemsAllowed" : 1, // Integers for
 "sldTouchStartX" : 0, // slider control
 "sldStartFlag" : 0, //
 "sldEndFlag" : 0, //
 "sldContentArray" : [ //Items in each slider
   {
   "imgPath" : "src of image",
   "imgLink" : "link",
   "descrText" : "Description text, may be HTML"
   }, ...
   ]
}

Each item in sldsObjs[] represents a slider.

This structure must be set before calling the scripts that generate the sliders, usually by your server-side code.

Once you're done with the steps above, to generate the sliders call renderAllSliders().
It could de linked to the window.onload event


In case windoww resizes, you can call adjustAllSliders() to make sliders fit in the screen in the correct way.
Again, this one should be linked to window.onresize event.

You can generate as much sliders as you want, at any place of your page.
