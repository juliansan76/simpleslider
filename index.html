HTML that sets a slider in your page

Firstly, you have to include simpleslider.css and simpleslider.js

Paste this html code at any place of your page:

<div id="sld0" class="sld-cluster">
</div>

Where sld0 is the id of the first slider. If threre's more than one, set their ids as sld0, sld1, sld2,.. and so on.

The data

There's a structure of data called that you must set up to provide the scripts that generate the sliders. Here's an example:

{ "sldItemWide" : 120, // width of each item in px
 "sldItemPosition" : 0, //
 "sldItemsAllowed" : 1, // Integers for
 "sldTouchStartX" : 0, // slider control scripts
 "sldStartFlag" : 0, //
 "sldEndFlag" : 0, //
 "sldContentArray" : [] //Slider elements array
}

This object is an item in the array sldsObjs[], and represents a slider. If you want to have more than one slider, just add elements to this array.

So, we have the object that represents a slider. Now, we have to provide the data that this slider will show.

That is, basically, filling sldContentArray[]. The elements of this array hold a path to an image, where it takes you, and a description of it

Here's an example:

{
"imgPath" : "img/img-01.jpg",
"imgLink" : "www.somewhere.com",
"descrText" : "Description text, may be HTML"
}

The scripts

Once you're done with the steps above, to generate the sliders call renderAllSliders().
This function will take every div tag with sld[i] id and make a slider from it. It could de linked to the window.onload event


In case main window resizes, you can call adjustAllSliders() to make sliders fit in the screen in the correct way.
Again, this one should be linked to window.onresize event.

You can generate as much sliders as you want, at any place of your page.
